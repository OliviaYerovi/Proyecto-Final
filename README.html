Comparacin y Anlisis del Gen rpoB en Escherichia coli y Campylobacter jejuni

Introduccin

El gen rpoB, que codifica la subunidad beta de la ARN polimerasa, se ha convertido en un gen candidato clave para los anlisis filogenticos y la identificacin de bacterias, especialmente al estudiar aislados estrechamente relacionados. La secuenciacin de rpoB permite una estimacin eficiente del contenido bacteriano de G+C%, el valor de hibridacin ADN-ADN y la identidad nucleotdica promedio (Adkambi & Raoult, 2009). De hecho, estudios comparativos han demostrado que rpoB supera al 16S rRNA en precisin para clasificar bacterias a nivel de especie (Case et al., 2007) Aunque rpoB se ha usado como marcador para identificar especies dentro del gnero Campylobacter (On et al., 1996), la comparacin especfica con E. coli puede proporcionar informacin valiosa sobre las diferencias evolutivas, estructurales y adaptativas entre estas bacterias.

Herramientas:

  • BWA BWA es un paquete de software para mapear secuencias de ADN con respecto a un genoma de referencia extenso, para instalar se puede seguir las instrucciones del desarrollador aqu (https://github.com/lh3/bwa)

  • SAMTOOLS Samtools es un conjunto de programas para interactuar con datos de secuenciacin de alto rendimiento. Se puede descargar desde su pgina (https://www.htslib.org/download/) o desde su github (https://github.com/samtools/)

  • MUSCLEes un programa de software utilizado para alinear secuencias mltiples de protenas o cidos nucleicos. La descarga se puede realizar desde el siguiente link (https://drive5.com/muscle/downloads_v3.htm)

  • MESQUITE Su objetivo es proporcionar un flujo de trabajo transparente que permita al usuario visualizar en tiempo real datos, rboles y anlisis para que pueda ver y comprender sus datos y resultados a medida que se procesan. Se puede descargar desde su repositorio (https://github.com/MesquiteProject/MesquiteCore/releases)

Estructura del Workflow

  1. Descarga de Datos de bacterias
    • Lecturas Fastq de E. coli y C. jejuni.
    • Genomas de referencia completos fastq ecoli_genome.fasta y jejuni_genome.fasta
  2. Preparacin del Entorno
    • Carga de herramientas necesarias como samtools o bwa
  3. Indexacin de Genomas
    • Usar los genomas descargados ecoli_genome.fasta y jejuni_genome.fasta
    • Preparacin para alineamiento con BWA y Samtools.
  4. Alineacin y Procesamiento
    • Alineacin de lecturas.
    • Conversin a formato BAM.
    • Ordenamiento e indexacin de archivos.
  5. Extraccin del Gen rpoB
    • Obtencin de las secuencias especficas del gen rpoB en ambos organismos.
  6. Alineamiento Mltiple
    • Fusin de las secuencias rpoB.
    • Alineamiento comparativo con MUSCLE.

Resultados Esperados

Una vez corrido el programa se espera obtener un archivo alineado que muestra las diferencias y similitudes entre las secuencias rpoB de ambas bacterias. Este archivo muscle_rpoB.fasta se ver en programa como Mesquite para el anlisis de la secuencia.

Esto es un ejemplo de como se vera los resultados en mesquite. Al abrir el archivo muscle_rpoB.fastase muestran las dos secuencias del gen rpoB en las bacterias E.coli y C.jejuni En el grfico cada fila corresponde a una de las dos secuencias genmicas analizadas, mientras que las columnas indican posiciones especficas del alineamiento. En cada celda se observa un nucletido (A, T, G o C), y los colores permiten distinguir visualmente las coincidencias y discrepancias entre las secuencias.

Estas diferencias a nivel de secuencia reflejan la divergencia evolutiva entre las dos bacterias, pese a que el gen rpoB conserva su funcin esencial. La identificacin de estas variaciones es til tanto para el anlisis filogentico como para estudios de resistencia a antibiticos, dado que mutaciones especficas en rpoB pueden estar relacionadas con resistencia a rifampicina. Este tipo de alineamiento, por tanto, no solo permite comparar la conservacin del gen entre especies bacterianas, sino tambin detectar regiones clave que podran tener implicaciones funcionales o clnicas.
